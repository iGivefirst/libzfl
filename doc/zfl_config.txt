zfl_config(7)
=============


NAME
----
zfl_config - load properties from configuration files


SYNOPSIS
--------
*zfl_config_t *zfl_config_new (void);*
*int zfl_config_method (zfl_config_t *config);*
*int zfl_config_test (void);*
*void zfl_config_destroy (zfl_config_t **config_p);*


DESCRIPTION
-----------

Loads a configuration file formatted in JSON or in ZPL format as defined
by rfc.zeromq.org/spec:4/zpl. Provides methods to navigate this data and
access property values. See zfl_device.c for examples of use.
